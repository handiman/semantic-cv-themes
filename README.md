# Semantic CV - Free themes

This repository contains the public collection of **free themes** used by:

- the Semantic‑CV CLI
- the upcoming site at [semantic.cv][semantic-cv]

The repository contains the **base theme classes** and the **default themes** that ship with the Semantic‑CV toolchain. Each theme follows the conventions of the Semantic‑CV ecosystem and is designed to produce clean, deterministic output from normalized CV data.

## Contents

- Base classes for defining Semantic‑CV themes
- A set of free themes included with the CLI
- The same themes available on the hosted renderer at [semantic.cv][semantic-cv]

## Not included
- CLI commands or tooling
- Normalization, analysis, or rendering logic from the core package
- Paid or premium themes
